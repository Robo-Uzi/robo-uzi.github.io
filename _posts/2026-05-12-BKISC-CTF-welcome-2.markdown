I put this into `input.txt` and run this python:
```python
import re

text = open("input.txt", encoding="utf-8").read()

# extract variation selector hex values
codes = re.findall(r'U\+(E01[0-9A-F]{2})', text)

# decode bytes
decoded = bytes(
    (int(c, 16) - 0xE0100) + 0x10
    for c in codes
)

print(decoded.decode())
```
