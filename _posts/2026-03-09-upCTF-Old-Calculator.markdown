**Title**: Old Calculator

**Category**: rev

**Author**: tmv11

**Description**: Found my old TI in a box of high school stuff. Weird, there's a program on here I don't remember installing. Wrap the code in `upCTF{...}`

I get the challenge file:
```shell
file PROG.8xp  
PROG.8xp: TI-83+ Graphing Calculator (assembly program)

strings PROG.8xp  
**TI83F*  
File generated by spasm  
PROG  
= CODE CHECKER =  
Enter code (CLEAR=reset):  
CORRECT! :)  
WRONG. Try again.  
aE> 2  
("O
```

The solve script will:
- Read the full `PROG.8xp` program into memory as raw bytes
- The `.8xp` file has a header then the Z80 program bytes. For TI-83+ ASM programs the first 55 bytes are header info.
- The encrypted flag bytes are stored at address `$9D98` in the Z80 program
- `$9D95` is the start of the program code in the file
- `$9D98` is where the 18 encrypted bytes of the “correct input” are stored
- File offset = `header_size + (program_address - program_start) = 0x4c + (0x9D98 - 0x9D95) = 0x4f`
- So `data[0x4f:0x4f+18]` grabs exactly the 18 encrypted key bytes
- The program uses a Galois Linear Feedback Shift Register (LFSR) to encrypt the bytes
- `key = 0xA5` is the initial seed of the LFSR.
- For each encrypted byte `b`:
    1. Step the LFSR to get the next key value
    2. XOR the encrypted byte with the key
- Result: `keycodes` now contains the raw scan codes corresponding to digits and letters.
- Converts all scan codes to characters

Solve script:
```python
#!/usr/bin/env python3

with open("PROG.8xp", "rb") as f:
    data = f.read()

# 18 encrypted expected bytes are at ($9D98)
# Theyre stored at file offset 0x4c + (0x9D98 - 0x9D95) = 0x4f
expected_enc = data[0x4f:0x4f + 18]

# Galois linear feedback shift register used by the comparison routine: SRL A; if carry: XOR $B8
def lfsr_step(x):
    return ((x >> 1) ^ 0xB8) if (x & 1) else (x >> 1)

# Decrypt: LFSR starts at 0xA5, step once per byte, XOR with encrypted byte
key = 0xA5
keycodes = []
for b in expected_enc:
    key = lfsr_step(key)
    keycodes.append(b ^ key)

# Map TI-83+ scan codes to characters
def keycode_to_char(k):
    if 0x8E <= k <= 0x97: return chr(ord('0') + k - 0x8E)
    if 0x9A <= k <= 0xB3: return chr(ord('A') + k - 0x9A)
    return '?'

flag = ''.join(keycode_to_char(k) for k in keycodes)
print(f"upCTF{{{flag}}}")
```
