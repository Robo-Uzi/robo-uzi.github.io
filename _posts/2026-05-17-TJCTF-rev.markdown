**Category**: rev

The password is hardcoded: `exposeTheNegative`

Once the correct password is passed to the program it will create the file `flag.png` and decrypt an embedded blob by XORing it with the bytes from the password.

I find the address of the encrypted blob and extract it:

Then I run this python:
```python
pw = b"exposeTheNegative"
enc = open("enc.bin","rb").read()

out = bytearray()

for i in range(len(enc)):
    idx = (i + ((i * 0xf0f1) >> 0x14) * -0x11) & 0xffff
    out.append(enc[i] ^ pw[idx % len(pw)])

open("flag.png","wb").write(out)
```
